# SafeFood_Project

안전 먹거리를 위한 사용자에 대한 알러지 정보, 열량, 등을 표시해 바른 먹거리 습관을 만들어주는 웹페이지.


----------------------------------------------------------------------------
## Skill
- XML Parsing
- JSP
- Spring
- MVC Pattern
- Mybatis
- Vue.js
- jQuery
- Bootstrap


## 개발환경
- IDE : Eclipse-photon
- Database : MySQL 8.0
- Java version : 1.8
